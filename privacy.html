Политика конфиденциальности

для Discord-бота Argus_Bot

Последнее обновление: 10.07.2025

1. Общие положения

Настоящая Политика определяет порядок обработки данных пользователей Discord-бота Argus_Bot (далее — "Бот"). Используя Бота, вы соглашаетесь с условиями этой Политики.

Важно: Бот не хранит содержимое ваших сообщений постоянно. Данные используются исключительно для работы функций модерации и автоматизации серверов Discord.

2. Права доступа бота

Важно: Бот отображается на Top.gg и проходит регулярную проверку модерацией платформы.

3. Собираемые данные

3.1. Автоматически собираемые данные

  • Идентификаторы (ID): пользователей, серверов, каналов, ролей
  • Метаданные сообщений: временные метки, авторство
  • Данные модерации: факты нарушений, действия администраторов

3.2. Временное хранение сообщений

Сообщения хранятся в базе данных (далее - "БД"), не более 10 дней для:

  • Команд модерации (!ban, !mute и т.д.)
  • Автоматической фильтрации нарушений
  • Обеспечения работы функций Бота
  • Истории сообщений и логирования

4. Хранение и защита данных

  • Место хранения: Сервер на территории РФ (хостинг Timeweb)
  • Сроки хранения:
    • В БД сообщений — до 10 дней
    • Журналы модерации — 30 дней
    • Настройки серверов — до удаления Бота
  • Меры защиты: Шифрование передаваемых данных, ограниченный доступ

5. Использование сторонних API

Важно: Используя функции бота, связанные с этими сервисами, вы соглашаетесь с их условиями предоставления данных.

Геоданные (ipinfo.io)

Для определения примерного местоположения (страны и города) по IP-адресу используется сервис ipinfo.io. Передаются только:

  • IP-адрес пользователя
  • Тип устройства

Данные не сохраняются в нашей базе и обрабатываются согласно политике ipinfo.io.

Spotify

  • Только публичные данные (названия треков, альбомов)
  • Не требует авторизации пользователя
  • Соответствует политике Spotify

  • Данные используются согласно политике Top.gg

  • Анонимизация данных перед обработкой

VirusTotal API

Для проверки файлов и ссылок на безопасность администраторы серверов могут использовать команду #!vtcheck, которая взаимодействует с VirusTotal.

Как это работает:

  • Файлы: временно загружаются в VirusTotal для анализа (не сохраняются после проверки)
  • URL-адреса: передаются в хешированном виде (SHA-256)
  • Данные не хранятся в базе бота после проверки
Важно: Проверка доступна только администраторам и модераторам. Отправляя файл или ссылку, они соглашаются с их передачей в VirusTotal в соответствии с политикой конфиденциальности VirusTotal.
